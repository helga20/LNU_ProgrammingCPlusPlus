Клас Pen моделює ручку.

Властивості:
1. Колір чорнила  задано константою для виведення на консоль
2. Тип чорнила    char type G-gel, I-ink, O-oil, P-paste
3. Запас чорнила  int length - кількість літер
4. Готовність     bool ready
--------------
5. Товщина пера
6. Ціна
7. Виробник
8. Конструкція
...

Дії:
1. Увімкнути
2. Вимкнути   ==> ready
3. Написати(рядок) ==> length, ready
4. Замінити чорнило ==> color, type, length
5. ...

Для того, щоб ручка справді моделювала написання різними кольорами, використано додаткові засоби керування консоллю.
Статичне поле Pen містить посилання на вікно консолі, статичний метод Init налаштовує це посилання і фон консолі.

Метод write працює, поки є чорнило. Про закінчення чорнила сигналізує виняток - ціле число, кількість виведених букв.

Конструктори і деструктор повідомляють про свою роботу.
